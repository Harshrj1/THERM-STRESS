🧠 Stress Detection using Thermal Imaging & Physiological Signals


🚀 Overview

This project presents a contactless stress detection system that leverages facial thermal imaging and physiological signals to analyze human stress responses. It evaluates changes across baseline, stress, and recovery phases and predicts a stress risk score using a deep learning model.


✨ Key Features


1.Contactless stress detection (no wearable devices required)

2.Multimodal analysis combining thermal and physiological data

3.CNN-LSTM model for spatial and temporal pattern learning

4.Explainable AI (SHAP, attention) for model interpretability

5.Outputs a quantitative stress risk score




🛠️ Tech Stack

1.Python

2.OpenCV (image processing)

3.NumPy (data handling)

4.TensorFlow / PyTorch (deep learning)

5.Matplotlib (visualization)




⚙️ Methodology

1.Capture facial thermal video

2.Segment key facial regions

3.Extract thermal and physiological features (temperature, heart rate, respiration)

4.Combine features into a unified representation

5.Analyze using a CNN-LSTM model

6.Apply Explainable AI techniques

7.Generate final stress risk score



🧪 Experimental Setup

The system operates in three phases:

Baseline: Normal relaxed state

Stress: Induced stress condition

Recovery: Return to normal state



📈 Applications

1.Healthcare monitoring

2.Mental stress analysis

3.Clinical screening

4.AI-based research systems
